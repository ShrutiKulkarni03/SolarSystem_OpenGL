Interactions:
> Use arraow keys to navigate.
> Use numpad keys 1-9 for planets information.
   Numpad1 - Mercury
   Numpad2 - Venus
   Numpad3 - Earth
   Numpad4 - Mars
   Numpad5 - Jupiter
   Numpad6 - Saturn
   Numpad7 - Uranus
   Numpad8 - Neptune
   Numpad9 - Pluto
> Use Mouse Left Mouse Button to click on Planets for planet information.
> Use key 'R' to reset the solar system
